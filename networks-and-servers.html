What Is a Network?

A network is a collection of computers, devices, and systems that are linked together to share resources and communicate.

Types of Networks:

  • LAN: Local Area Network (e.g., office networks)
  • WAN: Wide Area Network (e.g., the internet)
  • MAN: Metropolitan Area Network
  • Wireless Networks: Wi-Fi

Key Components:

  • Routers
  • Switches
  • Firewalls

What Is a Server?

A server is a specialized computer or software system that provides data, services, or programs to other computers over a network.

Common Functionality of server networks:

  • store and share your files;
  • share a single internet connection between all your devices;
  • allow staff to access files when out and about through a virtual private network (VPN);
  • centralise printing, so the server manages print jobs and lets you share printers;
  • host an intranet (a kind of private website holding important information about your business).
  • run networked applications and software;

In the past, a server was essential. However, most businesses use cloud computing services instead

Client/Server Network

A Client server network includes many clients or workstations that are connectedto at least one central server. most of the data and the applications get installed on a server

Example: When you access your work email, your device connects to a mail server via the network to retrieve and send messages.

WI-Fi USB Dongle

3G routers allow to connect USB dongle to the router just like how we connect cable connection to router. User can into connect USB dongle as internet connection to the router and use the internet over Wi-Fi on multiple devices.

The setup varies from D-Link to Net gear and Tata Photon to Reliance net connect. We are listing the general process.

  • Connect the 3G dongle in USB slot.
  • Connect the router with LAN wire that you get with the router. Router cable output wire goes as input wire to Laptop.
  • Open in the browser the link http://192.168.1.1/ similar to it. Check your router installation guide on more details.
  • You can see router configuration page and specify your ISP provider (example: Tata photon)
  • Enter required fields and set a password.

How Networks and Servers Work Together

Servers rely on networks to deliver services. Networks depend on servers for core functions like authentication and data hosting.

Example: When you access your work email, your device connects to a mail server via the network to retrieve and send messages.
